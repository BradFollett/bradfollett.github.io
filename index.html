Top 5 Detox/Wellness Vacations Slideshow

My "Top 5 Detox/Wellness Vacations" slideshow is a Java-based application created as part of my coursework in my CS-250: Software Development Lifecycle course that I completed in June 2023. This artifact represents my ability to develop a comprehensive software project that combines both functional and aesthetic elements, reflecting a real-world application of the software development lifecycle.

This project showcases my proficiency in Java programming, specifically in building graphical user interfaces (GUIs) using Java Swing. This artifact features a slideshow presentation that allows users to navigate through different slides showcasing the top 5 wellness/detox destinations. I utilized CardLayout to manage the different slides and implemented custom navigation controls to enhance user interaction. Also, I integrated an automatic slide transition feature, where the slides change every five seconds, offering a seamless and engaging experience for the user.

The most significant enhancement I made to this project involved updating the navigation system. Initially, the slideshow relied solely on mouse clicks for navigation, which could disrupt the user experience. To address this, I added keyboard navigation using the left and right arrow keys, allowing users to switch between slides more intuitively. This improvement not only made the application more user-friendly but also demonstrated my ability to consider and implement user-centric design principles in software development.

Overall, this artifact is a testament to my skills in Java programming and GUI development. It aligns with the course outcomes by demonstrating my ability to design and implement software solutions that are both functional and visually appealing. The enhancements I introduced reflect my commitment to continuous improvement and my ability to refine a project to meet high standards of usability and user satisfaction.
